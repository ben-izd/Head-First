DoE (Design of Experiments)

ابتدا باید آزمایش‌ها تولید شوند. فعلاً از ۳ روش برای تولید آزمایش استفاده خواهیم کرد که بطور پیش فرض متلب با بسته آمار و یادگیری ماشینی از آنها پشتیبانی می‌کند. برای اطمینان از نصب این بسته کد زیر را اجرا و لیست نمایش داده شده را بررسی کنید
First we have to generate our experiments. Currently, there are 3 way to generate the experiments, which Matlab support all of them through Statistics and Machine Learning Toolbox. If you are not sure, this toolbox is installed, run the following code, read the list.
ver
اگر بسته ذکر شده را مشاهده نکردید، قادر به اجرای کدهای ذکر شده در ادامه این مطلب نخواهید بود
If you don't see the mentioned toolbox, you can not execute the code below.

DoE-1 (Full Factorial)

از تابع ذکر شده برای تولید این نوع آزمایش استفاده کنید. این تابع یک بردار از اعداد را بعنوان پارامتر پذیرفته که تعداد اعداد موجود در بردار معادل تعداد پارامترهای آزمایش و خود اعداد بیانگر تعداد سطوح آن پارامتر جهت تولید آزمایش هستند
برای مثال برای تولید آزمایش برای ۳ پارامتر هریک با ۴ سطح می‌توانید به روش زیر عمل کنید
Use fullfact function to generate the experiments. It'll accept a vector which number of elements corresponds to number of paramters and each number represent the number of level for that parameter.
For example for generating experiments for 3 paramters, each with 4 level you should execute the following code:
doe = fullfact([4 4 4])
doe = 64×3
1 1 1 2 1 1 3 1 1 4 1 1 1 2 1 2 2 1 3 2 1 4 2 1 1 3 1 2 3 1

DoE-2 (Central composite circumscribed)

از تابع ذکر شده در متلب استفاده کنید. ورودی اول این تابع تعداد پارامترهای آزمایش را مشخص می‌کند. بطور پیش‌فرض آزمایش با نوع محیطی تولید می‌شود که نوع مدنظر است. برای مثال برای تولید آزمایش برای ۳ پارامتر می‌توانید به روش زیر عمل کنید
Use ccdesign function to generate the experiments. It can accepts one parameter which is the number of parameters. By defulat it will generate 'Circumscribed' type, so we don't have to change any options. For example, assume you have 3 parameters:
doe = ccdesign(3)

DoE-3 (Latin Hypercube)

از تابع ذکر شده استفاده کنید. این تایع دو ورودی دارد، اولی تعداد آزمایش و دومی تعداد پارامتر می‌باشد. برای مثال جهت تولید ۵ آزمایش برای ۳ پارامتر می‌توانید از کد زیر استفاده کنید
For latin hypercube, Matlab has lhsdesign which accepts two parameters, first is the number of samples and second is the number of columns (or number of parameters). For example, assume we have 2 variables and we want to generate 5 samples:
doe = lhsdesign(5, 3)
doe = 5×3

Rescaling

خروجی کد های بالا، در محدوده پیش فرض تولید شده که مناسب فعالیت ما نیست، برای حل این موضوع می‌توانید اعداد تولید شده را دستکاری نمائید
دستکاری مجزای هر ستون به شکل راحت‌تر انجام می‌شود. بطور معمول هر ستون محدوده خود را دارد ولی در مثال زیر تمام پارامترها باید بین ده تا بیست قرار گیرند. برای دستکاری اعداد به کد زیر توجه فرمائید
Since the above functions generate experiments without specifying a range, here we'll fix the issue by rescaling the design.
In our design it's simpler to rescale each column seperately. Normally each column has its own range. For simplicity we assume all of 3 parameters have the same range of 10 to 20, the following code can be used:
lower_bound = 10;
upper_bound = 20;
 
doe(:, 1) = rescale(doe(:,1), lower_bound, upper_bound);
doe(:, 2) = rescale(doe(:,2), lower_bound, upper_bound);
doe(:, 3) = rescale(doe(:,3), lower_bound, upper_bound);

Simulation

برای اجرای آزمایشات تولید شده، تابع خود را پیدا کنید. هر سه تابع از ورودی به شکل ماتریس پشتیبانی می‌کند که هر ستون نمایانگر یک پارامتر می‌باشد. صرفاً آزمایشات دستکاری شده خود را به عنوان ورودی به تابع دهید و خروجی را دریافت کنید
Use the coressponding function for your group. Each function accepts a matrix where each column represent a parameter. Simply use your rescaled experiments as input and recieve the output.

Sim-1 (Rosenbrock equation)

function result = rosenbrock(x)
result = (-1+x(:,1)).^2+100*((x(:,1).^2-x(:,2)).^2)+(-1+x(:,2)).^2+100*((x(:,2).^2-x(:,3)).^2);
end

Sim-2 (Gas turbine)

function result = gas_turbine(data)
% data should have 3 columns:
% - 1st column is T_amb
% - 2nd column is B
% - 3rd column is TIT
 
m_air = 90;
c_p = 1.01;
k = 1.4;
n_c = 0.89;
n_t = 0.92;
n_el = 0.98;
n_m = 0.98;
 
T_amb = data(:,1);
B = data(:,2);
TIT = data(:,3);
 
W_c = m_air * c_p * T_amb .* (1 - B.^((k-1)/k)) ./ n_c;
Q_1 = m_air * c_p * (TIT-T_amb.* (1+(B.^((k-1)/k)-1) ./ n_c));
W_t = m_air * c_p * TIT .* (1 - 1./(B.^((k-1)/k))) * n_t;
 
P_net = n_m * n_el * (W_t + W_c);
 
result = P_net ./ Q_1;
end

Sim-3 (Wind turbine)

function result = wind_turbine(data)
% data should have 3 columns:
% - 1st column is rho
% - 2nd column is A
% - 3rd column is v
 
rho = data(:,1);
A = data(:,2);
v = data(:,3);
 
zeta = interp1(1:25,[0, 0.12, 0.29, 0.4, 0.43, 0.46, 0.48, 0.49, ...
0.5, 0.5, 0.44, 0.36, 0.29, 0.23, 0.19, 0.15, 0.13, 0.11, ...
0.09, 0.08, 0.07, 0.06, 0.05, 0.05, 0.04],v);
 
result = 0.0005 * zeta .* rho .* A .* (v.^3);
end
