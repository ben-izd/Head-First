Task 3 - Monte Carlo Simulation

اکنون باید با ۳ اندازه نمونه - شبیه‌سازی مونت کارلو را اجرا و میانگین، انحراف معیار محاسبه کرده و هیستوگرام داده را رسم کنید
Now we have to run 3 simulation each with different number of samples (10, 10^3, 10^5), compute the mean and standard deviation and plot its histogram.
% Redfine RV to support vectorization
PV = @(A,r,PR,GHI) A .* r .* PR .* GHI;
 
GHI_dist = GHI_dist3;
 
GHI10 = random(GHI_dist, [10 , 1]);
GHI1000 = random(GHI_dist, [1000 , 1]);
GHI100000 = random(GHI_dist, [100000, 1]);
براساس تکلیف گروه خود باید پارامترهای متغیرها را تعریف کنید. خالی بودن ستون سیگاما متغییر بیانگر عدد بودن آن وعدد بودن ستون بیانگر توزیع بودن متغییر است. در مثال ما متغییر پی‌آر عدد است درحالیکه که متغییر آر توزیع است (بسته به تکلیف باید کد زیر را دستکاری نمائید – اگر متغییری توزیع است باید ۳ متغییر بسازید که هر کدام شامل ده – هزار و صد هزار نمونه از آن توزیع هستند – بعد باید کد زیر عبارت اجرا (به انگلیسی) دستکاری شود)
You have to change the following paramters according to what has been asked. If the σ column is empty, that parameter is a number, if not, it's a distribution. In our example, PR is a number while r is a distribution. If a parameter is a distribution, you have to create 3 variables just like r and used them in the % Evaluate part.
 
% Define Parameters
A = 10;
PR = .50;
r_dist = makedist('Normal', 'mu', 0.1, 'sigma', .01);
 
r10 = random(r_dist, [10 , 1]);
r1000 = random(r_dist, [1000 , 1]);
r100000 = random(r_dist, [100000 , 1]);
 
 
% Evaluate
result10 = PV(A, r10 , PR, GHI10 );
result1000 = PV(A, r1000 , PR, GHI1000 );
result100000 = PV(A, r100000, PR, GHI100000);
 
disp(array2table([mean(result10) std(result10); ...
mean(result1000) std(result1000); ...
mean(result100000) std(result100000);], ...
'VariableNames',{'mean','std'}, ...
'RowNames',{'10','10^3','10^5'}));
mean std ______ _______ 10 4.452 0.59977 10^3 4.2145 0.49268 10^5 4.2142 0.49085
histogram(result100000, 'Normalization', 'pdf');
 
hold on;
 
histogram(result1000, 'Normalization', 'pdf');
histogram(result10 , 'Normalization', 'pdf');
 
legend({'100,000 Samples', '1,000 Samples', '10 Samples'});
hold off;
در مثال ما، نمونه با اندازه هزار دقت کافی را داراست و به عنوان اندازه برتر انتخاب می‌شود
حالا باید مراحل قسمت دوم را تکرار نموده و ۳ توزیع را به خروجی نمونه برتر (هزار تایی در این مثال) فیت کنیم – (در این مثال ما از همان سه توزیع قبل استفاده خواهیم کرد – شما می‌توانید توزیع متفاوتی انتخاب کنید)
In our case, 1000 samples had enough accuracy, so we'll pick this number as the best sample size.
Now we have to repeat the task 2 for our result1000. Just like before you have to pick 3 distribution, here we'll pick the same set of distribution as before (you can change it if you want).
result_dist1 = fitdist(result1000, 'Normal' );
result_dist2 = fitdist(result1000, 'Lognormal');

disp(array2table([result_dist1.ParameterValues; ...

Task 5 - MSPE Analysis

در این مرحله باید با کمک فایل اکسل ذکر شده آزمایش اِم اِس پی ایی را جهت ارزیابی اندازه نمونه برتر مرحله قبل انجام دهیم. عدد هزار در کد پائین را با اندازه برتر خود عوض کنید) – اجرای کد زیر فایلی با ۴ ستون و هزار سطر تولید می‌کند – فایل را باز کرده و داده‌ها را کپی کنید، سپس به سراغ فایل تحلیل رفته، محدوده ذکر شده را پاک کرده و در بخش ران (سلول ب۴) داده‌ها را جایگذاری کنید
For this section, you have to use the MSPE_2023.xlsx file which you can access through Teams. Here we want to check wether our best sample size in task 3 is optimal or not. Replace 1000 with your best sample size. The following code will create a csv file which in our case has 1000 rows and 4 columns. Open the file and copy the data, then open MSPE_2023.xlsx, clean B3:E10002 and paste it under the Run (Select B4 cell and paste).
writematrix(random(result_dist1, [1000, 4]), 'mspe_data.csv');
بعد از جایگذاری به سراغ برگه ذکر شده بروید. براساس داده شما ممکن است تنظیمات پیش فرض مناسب نبوده و نیاز به تغییر داشته باشد
بصورت پیش فرض نمودار ده هزار داده را نمایش می‌دهد. اگر اندازه برتر شما کمتر است، به بخش ذکر شده رفته و داده‌های نمودار را دستکاری نمائید
برای تغییر ناحیه نمایش، روی نمودار عمودی دابل کلیک کرده و مقدار بیشینه را تغییر دهید
سپس نمودار موجود را در گزارش خود قرار دهید. برای مثال ما شکل زیر را در نظر بگیرید

By default the plot will plot 10,000 elements, if you have less, first select the plot, then go to Chart Design > Select Data. On the left panel select each series change the range to your specific size.
To change the view, double click on the vertical axis (left numbers), change the amount of maximum.
